1:1 공 피하기

  • 2021. 11 - 2021. 12
  • 1인 프로젝트
  • PyCharm
  • Python
  • 데이터 저장 없음


파이썬의 소켓 통신을 이용해 구현한 서버와 클라이언트의 1:1 공 피하기 게임입니다. 서버 사용자가 게임 통신을 시작하면 클라이언트 사용자가 접속해 플레이할 수 있도록 제작된 프로그램이며, 심심할 때 친구와 플레이할 수 있는 간단한 게임을 만들고 싶어서 1:1 공 피하기를 프로젝트의 주제로 선정하게 되었습니다. 게임의 결과를 따로 저장할 필요성을 느끼지 못해 데이터 저장은 구현하지 않았으며, 같은 화면을 출력할 수 있도록 데이터의 동기화를 진행했습니다.

✔주요 기능


방향키를 이용한 캐릭터 움직임, 캐릭터 점프, 1:1 통신, 2대의 컴퓨터의 데이터 동기화, 공 충돌 처리, 승리/패배 판단

✔GitHub


https://github.com/Verygood-wony/avoid_the_ball

✔사용 기술


PyGame, 소켓 통신, 다중 스레드

✔느낀점


1. 단기간에 소켓 통신을 이용한 프로그램을 작성하며 네트워크 통신에 대해 더욱 깊게 알게 되었습니다.

2. PyGame을 이용하며 Python의 다양한 라이브러리를 몸소 느낄 수 있었으며 Python이 다양한 용도로 사용되는 이유를 알게 되었습니다.

3. TCP/IP와 UDP의 차이를 알게 되었고 어떤 상황에서는 어떤 통신 방식이 유리할지 고민하게 되는 계기가 되었습니다.
