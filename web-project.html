자기소개 HTML

  • 2020. 10 - 2020. 12
  • 1인 프로젝트
  • Sublime Text
  • HTML5, CSS, JS
  • 데이터 저장 없음


웹 프로그래밍 과목에서 진행한 자기소개 웹 페이지 제작 프로젝트입니다. 간단한 자기소개 및 자신이 꾸미고 싶은 방식으로 꾸며서 제작하였습니다. 좋아하는 음악을 넣었으며, 간단한 자기소개 및 프로그래밍 능력을 수치화하여 표시하였습니다. 웹 프로그래밍 능력을 키우기 위해 다양한 JavaScript를 넣어 웹 페이지를 다양하게 꾸며보는 경험을 하였습니다.

✔주요 기능


상단 내비게이션 바 고정, 스크롤 이벤트 함수, 핸드 오버 처리 등

✔느낀점


1. 첫 웹 페이지를 제작했다는 것만으로도 의미가 있었지만 그 과정에서 JavaScript와 CSS 등 웹 페이지를 꾸밀 수 있는 방법을 알게 되었다는 점에서 더욱 의미가 있었습니다. 또한, 포트폴리오 홈페이지를 만들 때 이 프로젝트 진행 경험이 많이 도움이 되었습니다.

2. 웹 페이지 제작 능력을 상승시킬 수 있었으며 HTML5, CSS, JavaScript의 기초를 배울 수 있었습니다.

3. 온라인상에 존재하는 다양한 HTML 템플릿을 보며 사용자 경험과 가장 알맞은 UX/UI에 대해서도 생각해 보는 계기가 되었습니다.
