Jump King

  • 2020. 04 - 2020. 06
  • 1인 프로젝트
  • Eclipse
  • JAVA
  • txt


방향키와 스페이스 바를 이용해 원하는 목표지점까지 점프를 하여 도달하는 게임입니다. 기존에 존재하는 Jump King 게임을 오마주한 게임이며, 혼자 즐길 수 있는 게임을 제작하고 싶어 프로젝트를 제작하게 되었습니다. 게임의 클리어 기록은 txt 파일에 기록되며 4스테이지에서 밑으로 떨어질 경우에는 랜덤으로 1~3스테이지로 이동하게 됩니다.

✔주요 기능


방향 전환, 스페이스 바 입력 시간에 따른 점프 높이, 충돌 처리, 게임 클리어 정보 저장, 스테이지 이동

✔GitHub


https://github.com/Verygood-wony/jump_king

✔느낀점


1. Java 언어를 사용해 게임을 제작했다는 것만으로도 의미가 있었지만 그 과정에서 게임에 필요한 물리적 표현과 수학적 지식에 대해 알게 되었다는 점에서 더욱 의미가 있었습니다. 또한, 게임을 설계하는 방법과 이를 효율적으로 구현하는 방법을 알게 되었습니다.

2. Java 프로그래밍 능력을 상승시킬 수 있었습니다.

3. Java Swing 컴포넌트에 대해 더욱 자세하게 알 수 있게 되었습니다.

4. 게임을 제작해 보며 다중 스레드와 객체 프로그래밍에 대해 고민해 보는 시간을 가질 수 있었습니다.
